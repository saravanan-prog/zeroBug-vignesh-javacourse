JAVA variables
===============

    =>  In Java, variables are containers that hold data values.

    => Every variable has
           Syntax :
              variable_type variable_name;
                (or)
              variable_type variable_name = value;

    => Declaring Variables

            int age;              ---> Stores the whole values
            double price;         ---> decimal values
            char grade;           ---> Single Character
            boolean isActive;     ---> true or false Flag values
            String name;          ---> This is class type "Stores seqeuence of characters"

    => Initializing Variables

            int age = 20;
            double price = 19.99;
            char grade = 'A';
            boolean isActive = true;
            String name = "Alice";


Variable Scope
==============
    1. Local Variables
    -------------------
         =>   Declared inside methods
         =>   Must be initialized before use


       Example
       -------
            void method() {
                int x = 10; // local variable
            }


    2. Instance Variables (non-static fields)
    -----------------------------------------
       => Declared inside a class but outside methods
       =>  Each object gets its own copy

           class Person {
               String name = "saravanan"; // instance variable

               public static void main(String args[]){
                 Person obj = new Person();
                 System.out.println("name ===>" + obj.name);
               }
           }

    3. Static Variables (class variables)
    -------------------------------------
       => A static variable is a variable that belongs to the class, not to the objects.

            class Person {

                  static String name = "Saravanan"; // static variable

                  public static void main(String args[]){

                        System.out.println("name ===>" + name);
                  }
            }

    4. Final Varaibles [Constant]
    ===============================
        => values cannot be mutable
        => final varaibles it can declared final keyword with caps.


         class Person {

             final String dbName = "mysql";
             final String paymentType = "Razor-pay";


            public static void main(String args[]){

                System.out.println("name ===>" + name);
            }
         }
